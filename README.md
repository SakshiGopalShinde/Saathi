## Project Vision ‚ú®

Saathi is built to address the growing mental health challenges among students by making support:

- Accessible  
- Confidential  
- Culturally relevant  
- Proactive  

Instead of fragmented solutions, Saathi provides an integrated platform that combines AI-based first aid with professional and peer support.

---

## Motivation

### The Unmet Need
Mental health concerns among students are increasing rapidly, while support systems remain fragmented and generic.

### The Accessibility Gap
Traditional counselling often faces barriers such as stigma, limited availability, and scheduling conflicts.

### The Digital Disconnect
Many mental health apps are Western-centric, subscription-based, and disconnected from campus counsellors or local language support.

### The Need for Proactive Insight
Institutions lack real-time anonymized data to detect stress trends and intervene early.

## Data Flow

1. A student interacts with the AI chatbot.  
2. The AI analyzes the conversation and suggests coping strategies or screening tools.  
3. If high-risk indicators appear, the system prompts appointment booking.  
4. Students can book counsellor sessions or explore the resource hub.  
5. Anonymized data feeds into the admin dashboard for institutional insights.

## System Architecture Details

### Frontend (User Side)
- Responsive web and mobile app using React.js or Flutter  
- Handles chatting, booking appointments, and accessing resources  
- Supports text/voice input and multilingual content  

### Backend (Server Side)

**AI & Logic**
- Python backend using TensorFlow or PyTorch  
- Sentiment analysis and clinical screening  

**Core Services**
- Node.js backend managing APIs  
- Booking system, peer forum, and resource hub  

**Databases**
- PostgreSQL or MySQL for structured data  
- MongoDB for chat logs and journals  

**Admin & Analytics**
- Processes anonymized data for institutional dashboards  

**Cloud & Security**
- Hosted on AWS, GCP, or Azure  
- Docker & Kubernetes for scalability  
- Encrypted storage and secure authentication
